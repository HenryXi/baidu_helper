# 百度网盘油猴脚本

百度网盘网页版和客户端都不好用，这个脚本可以帮助自己快速浏览网盘中的资源

## 使用方法

0. 添加脚本到油猴插件中
1. 登录百度网盘网页版
2. 打开浏览器调试模式找到api请求地址类似这样：https://pan.baidu.com/api/list?……
3. 在浏览器中打开上面的地址就可以了
4. 上面连接中dir字段就是对应的网盘路径
